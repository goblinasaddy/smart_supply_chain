## Data Flow

1. **Data Layer** generates/fetches weather, traffic, shipment data
2. **Coordinator Agent** receives data, dispatches to specialist agents
3. **Shipment Agent** updates shipment statuses
4. **Risk Agent** computes risk scores using ML model
5. **Routing Agent** (conditional) reroutes high-risk shipments
6. **Explainability Layer** generates LLM explanations
7. **API Layer** serves results to frontend
8. **Dashboard** visualizes everything in real-time
